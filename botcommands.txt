forecastcity - получить прогноз по выбранному городу
forecastgeo - получить прогноз по выбранной геолокации
prevforecastcity - получить прогноз по предыдущему городу
prevforecastgeo - получить прогноз по предыдущей геолокации
forecastcityhistory - посмотреть историю запросов по городу и выбрать один из них
forecastgeohistory - посмотреть историю запросов по геолокации и выбрать один из них
forecastcityclear - очистить историю запросов по городу
forecastgeoclear - очистить историю запросов по геолокации
toggledaily - включить/выключить ежедневный прогноз (по умолчанию выключен)
setdailytime - установить время ежедневного прогноза (по умолчанию 9:00)
setdailycity - установить город ежедневного прогноза (по умолчанию Москва)
